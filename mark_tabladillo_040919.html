Mark Tabladillo

Tuesday, April 9th, 2019

Deep Learning Architectures with Azure

Microsoft Azure offers several common configurations for deep learning. This presentation will outline the major options and differences -- and provide demonstrations that you can try now. The choices inlude the Deep Learning Virtual Machine, HD Insight (Hadoop), Azure Databricks, and Azure ML Service

Mark Tabladillo Ph.D. works for Microsoft as a Cloud Solution Architect for the US CTO Customer Success Team. He has a science doctorate from Georgia Tech. Based in Atlanta, GA, Mark provides cloud enterprise solutions for companies in the US for Microsoft's strategic clients. He is a chapter leader for the PASS Data Science virtual chapter. He tweets @marktabnet and connects with friends on LinkedIn.
